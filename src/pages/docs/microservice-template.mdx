title: Microservice Template
description: The template for the file used when creating a new microservice project using the `new:micro` command.
---

## Authorization Settings Properties

| Name       | Required | Description                                                         | Default                          |
| ---------- | -------- | ------------------------------------------------------------------- | -------------------------------- |
| [SolutionName](/docs/solution-name)    | Yes      | The name of the solution. | *None*   |
| [AddGit](/docs/git)    | No      | Settings for git in your project. | *true*  |
| [Microservices](/docs/microservice)       | Yes      | The microservices that you want to add to your project | *None*   |
| [Gateways](/docs/gateway)        | Yes      | A list of gateways to add to your project. | *None*   |

## Microservice Template Example
The example on the [new:micro command page](/docs/new-micro) covers how to set up a microservice template.
