---
title: Add Entity Template
description: The template for the file used when adding new entities to a web API using the `add:entity` command.
---

## Authorization Settings Properties

| Name       | Required | Description                                                         | Default                          |
| ---------- | -------- | ------------------------------------------------------------------- | -------------------------------- |
| [Entities](/docs/entities)        | Yes      | A list of entities to create database endpoints. | *None*   |

## API Template Example
The add entity template only needs the details for the new entity objects you want to add. You can find more details on the [add entity command page](/docs/add-entity).

```yaml
Entities:
  - Name: Supplier
    Properties:
      - Name: SupplierId
        IsPrimaryKey: true
        Type: int
        CanFilter: true
        CanSort: true
      - Name: Name
        Type: string
        CanFilter: true
        CanSort: true
      - Name: EmployeeCount
        Type: int?
        CanFilter: true
        CanSort: true
      - Name: CreationDate
        Type: datetime?
        CanFilter: true
        CanSort: true
      - Name: SupplierType
        Type: int?
        CanFilter: true
        CanSort: true
```
