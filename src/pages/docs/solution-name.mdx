## Solution Name
The solution name is just a simple string that will become the name of your API.

## Solution Name Properties
| Name          | Required | Description                                                  | Default |
| ------------- | -------- | ------------------------------------------------------------ | ------- |
| Solution Name | Yes      | A string that designates the name you want to use for your API. | *None*  |

## Solution Name Example
This example would create a solution with the name `VetClinic.Api`
```yaml
SolutionName: VetClinic.Api
```
